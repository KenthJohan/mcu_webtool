SQL Query Interface

Execute custom SQL queries against the DuckDB database. Write your query below and click "Execute Query" to see the results.

Example Queries (click to load):

SELECT * FROM parameters LIMIT 10
SELECT COUNT(*) as total_rows FROM parameters
SELECT page, COUNT(*) as param_count FROM parameters GROUP BY page ORDER BY param_count DESC
SELECT * FROM parameters WHERE page = 1
SELECT DISTINCT page FROM parameters ORDER BY page
